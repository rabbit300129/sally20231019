## 安裝 nodejs

https://nodejs.org/en

- 下載 LTS 版本並安裝
- 安裝完成後關閉 `vscode` 再開啟，確認環境有載入
- 開啟 `vscode` `cmd` 終端機
- 輸入 `node -v` 如有看到版本號表示安裝成功

## 安裝 firebase-tool

```shell
npm install -g firebase-tools
```

## 設定 firebase hosting

### 登入

```shell
firebase login
```

### 初始化

```shell
firebase init
```
